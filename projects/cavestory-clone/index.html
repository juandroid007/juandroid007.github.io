Cave Story Clone

This small project was a challenge in which I decided to create a technical clone of the famous game Cave Story (D≈çkutsu Monogatari), a masterpiece of the japanese indie developer Daisuke "Pixel" Amaya, developed in his free time over five years.

The intention I had was to explore the creation of my own engine, which is why the language I chose was C++, accompanied by the SDL2 library to handle graphics in a simple and universal way. To make the code as flexible as possible, I restricted myself to using only the standard C++ library (C++17, with the exception, obviously, of SDL2), opening the possibilities for cross-compilation.

In the end, I got a tile-based engine with the following characteristics:

  • Rectangle-based collision system, with composite rectangles and singles rectangles.
  • Tile-based maps.
  • Simple physics engine.
  • Game units.
  • Slope tiles.

The source code is here: github.com/juandroid007/cavestory-clone
